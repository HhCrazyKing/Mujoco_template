Legged Robotics Course Page

Instructor: Pranav Bhounsule

A first course on Legged Robotics. It is beneficial to have prior knowledge of undergraduate dynamics and programming, but these are also taught here. The structure of the course is to teach theory and supplement that with simple examples and computer code. Using MATLAB one can see how to code the physics, control, and animate. Using CoppeliaSim (a free robotics simulator for education purposes) one can see how to use a simulator to model and write control code. MATLAB 2021a (need symbolic toolbox, optimization toolbox) and CoppeliaSim v4.1.0 (July 21, 2020) were used.
