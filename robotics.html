About: A first course in robotics. The course emphasizes learning through extensive numerical simulations and animations. Mathematical modeling and theory is explained to build the necessary intuition to develop numeric code. MATLAB code provides examples on how to program the physics, develop controllers, and animate the system (see Downloads/Links: MATLAB links for tutorials). The YouTube playlist is here http://tiny.cc/robotics_f20. A free simulator for education, CoppeliaSim, is used for modeling and control code is written in Lua. MATLAB 2021 with symbolic and optimization toolbox and CoppeliaSim v4.1.0 (July 21, 2020) were used.
