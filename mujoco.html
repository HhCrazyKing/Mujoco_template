About:   This mini course aims to give you working knowledge of MuJoCo, rigid body dynamics and controls. The tutorials are based on using mujoco200 available at roboti.us. The software needs the file mjkey.txt which is also available on the website. A shortened link to this webste is http://tiny.cc/mujoco. The YouTube playlist is here: http://tiny.cc/mujoco2. The course is taught using C interface of MuJoCo. Prior knowledge of C/unix commands is not assumed. Everything is taught from scratch. All software used in this course is freely available. This includes MuJoCo 200, Atom Editor, C, make, Visual Studio (community edition) for Windows
